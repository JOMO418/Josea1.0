## ğŸ†˜ TROUBLESHOOTING

### **Problem: "Node.js is not installed"**
**Solution:**
1. Download Node.js from: https://nodejs.org
2. Install the LTS (Long Term Support) version
3. Restart your computer
4. Run `SETUP-FIRST-TIME.bat` again

### **Problem: "PostgreSQL is not running"**
**Solution:**
1. Press `Win + R`
2. Type: `services.msc` and press Enter
3. Find "PostgreSQL" in the list
4. Right-click â†’ Start
5. Run `START-JOSEA-PRESENTATION.bat` again

### **Problem: System won't start or shows errors**
**Solution:**
1. Close all command windows
2. Run `STOP-SYSTEM.bat`
3. Wait 5 seconds
4. Run `START-JOSEA-PRESENTATION.bat` again

### **Problem: Browser doesn't open automatically**
**Solution:**
- Manually open your browser and go to: `http://localhost:5173`
